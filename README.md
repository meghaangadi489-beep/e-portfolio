# 🚀 Engineering E-Portfolio — Megha Angadi

This repository contains my personal academic e-portfolio showcasing my skills, projects, and co-curricular experiences as an engineering student.

🔗 **Live Website:** [https://meghaangadi489-beep.github.io/e-portfolio/](https://meghaangadi489-beep.github.io/e-portfolio/)

---

## 🌟 Key Features

*   **Modern Responsive Design:** Crafted with custom styles, sleek gradients, Outfit and Inter Google fonts, and interactive micro-animations.
*   **Dynamic Typing Effect:** Integrated typewriter animation for current interests (Programming, AI exploration).
*   **Scroll-Progress Bar:** Real-time visual reading progression indicator.
*   **Contact Form:** Fully styled dynamic contact interface.
*   **Scroll-to-Top:** Seamless navigation control for desktop and mobile viewports.

---

## 📂 Portfolio Structure

*   **Home:** Quick overview and professional summary badge.
*   **About Me:** Background, academic goals, and tech interests.
*   **Skills:** Performance indicators representing skills like C Programming, Git/GitHub, TinkerCAD, and Prompt Engineering.
*   **Projects:** Focus areas including:
    *   *Digital Wallet System* (C Command Line Interface, encrypted ledger).
    *   *2D Graphics Editor* (Computer graphics rendering logic).
*   **Activities:** Academic timelines detailing Hardware Dissection Workshops, sewage treatment plant industrial study visits, and TinkerCAD Arduino simulations.
*   **Contact:** Direct email linkages and interactive contact submit layouts.

---

## 🛠️ Built With

*   **HTML5** — Semantic page architecture.
*   **CSS3 (Vanilla)** — Sleek dark-themed layout, glowing accents, and keyframe transitions.
*   **JavaScript (ES6)** — Dynamic animations, menu state toggle controls, and interaction handlers.
*   **FontAwesome Icons** — Visual indicators for key sections.

---

## 🚀 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/meghaangadi489-beep/e-portfolio.git
   ```
2. Navigate into the folder:
   ```bash
   cd e-portfolio
   ```
3. Open `index.html` in any web browser to view the application.
